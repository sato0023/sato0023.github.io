  4. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: The <i> tag is the odd one out because it affects the style of text, whereas the other three tags alter the function/placement of text (the <a> tag creates a hyperlink, the <p> tag creates a paragraph, and <span> is used to grup in-line elements to a document or provides a way to hook to a part of a text or part of a document).

  6. Qus: What element is used to create images? Name its attributes.

    Ans: The <img> tag defines an image in HTML, and src and alt are its attributes.

  7. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

    Ans: <b> tag is out of place as it bolds the font in HTML, but all the other three have to do with creating tables. The <td> tag defines a standard cell in an HTML table, the <tr> tag defines table rows, and the <table> tag defines an HTML table.
